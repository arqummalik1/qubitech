## ğŸš¢ Deployment (GitHub Pages)

This project is configured for one-click deployment to GitHub Pages.

1.  **Build and Deploy**
    ```bash
    npm run deploy
    ```
    *This script builds the project (`vite build`) and pushes the `dist` folder to the `gh-pages` branch.*

2.  **Configuration**
    -   `vite.config.js`: `base: '/qubit-tech/'`
    -   `package.json`: `"homepage": "https://arqummalik1.github.io/qubit-tech"`
